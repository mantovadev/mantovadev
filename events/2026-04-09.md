## Talk

- **Titolo:** Parla con i tuoi dati: accesso ai database in linguaggio naturale con Java
- **Speaker:** [Marco Belladelli](https://www.linkedin.com/in/marco-belladelli/) (Software Developer @ IBM)
- **Descrizione:**
Un viaggio pratico nel mondo Java su come sfruttare gli LLM per "parlare" ai database relazionali in linguaggio naturale, senza dover saper scrivere query o conoscere nel dettaglio la struttura dei dati. Vedremo come chiunque possa ottenere informazioni preziose e rilevanti al proprio contesto, esplorando anche un approccio RAG e la possibilità di esporre le funzionalità via MCP per integrarle con un team di agenti.
